# Contributing to Bhek Network Global

Welcome! We are looking for visionaries to help expand the Omni-Sovereign OS.

### How to Contribute
1. **Fork** the repository.
2. Create a **Feature Branch** (`git checkout -b feature/NewNode`).
3. **Commit** your changes with clear messages.
4. Open a **Pull Request**.

### Standards
- Keep the Glassmorphism aesthetic.
- All code must be compatible with Puter.js v2.
